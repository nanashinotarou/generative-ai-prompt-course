# 制作ガイドサイト 開発・デザイン基準 (DESIGN_STANDARDS.md)

このドキュメントは、研修教材サイトの品質を一貫して保つための基準を定義します。

## 1. ビジュアル・アイデンティティ (UI/UX)
- **Glassmorphism**: `backdrop-filter: blur(20px)` を多用し、透明感のあるカードを基本とする。
- **Bento Grid**: 情報をグリッド状に整理し、マウスホバーで浮き上がるアニメーション（`transform: translateY(-5px)`）を付与する。
- **階層化されたタイポグラフィ**: 見出しは `font-weight: 900`、フォントサイズは `clamp` を使用してモバイルフレンドリーにする。
- **プレミアムな配色**: 単色ではなく、`linear-gradient` や `radial-gradient` を背景やアクセントに多用する。

## 2. ゲーミフィケーション・インタラクティブ要素
- **Flip Cards**: 重要な概念はタップ/クリックで裏返るカード形式で提示する。
- **Quiz**: 選択肢を選んだ際のアニメーション、正誤判定後の詳細解説ボックスは必須。
- **Mission Checklist**: タスクを完了した際のチェックボックス、全完了時の「蝋封（Wax Seal）」スタンプアニメーション、コンフェッティ演出を実装する。
- **プログレスバー**: ページの読了状況やタスクの進捗を同期させ、学習者の達成感を醸成する。

## 3. 教材としての教育的深み
- **動画不要の教科書化**: 動画を観なくてもコアのエッセンスが学べるよう、詳細なテキスト解説、比較表、図解（CSS/SVGダイアグラム）を配置する。
- **Pro Tips / Point**: 現場で役立つ「コツ」を際立たせる情報ボックス（電球アイコン等）を設ける。
- **ワークフローの可視化**: 手順を `Step 1, 2, 3` のように番号付きで、図解とともに解説する。
